## Gist Viewer
> A tiny react and redux based app that shows a listing of gists for a github user.

### Setup

```bash
    # clone repository
    位 git clone https://github.com/umayr/gist-viewer
    # change directory
    位 cd gist-viewer
    # install dependencies
    位 yarn # or alternatively use npm
    # start app in developer mode
    位 yarn start
```
